# 📸 Insta Clone

A full-stack Instagram-inspired social media application that allows users to share posts, interact with others, and experience a modern social platform UI.

🌐 **Live Demo:** https://insta-clone-va4x.onrender.com/  
💻 **GitHub Repo:** https://github.com/vivekchanne06-web/Insta-Clone  

---

## 🚀 Features

- 🔐 User authentication (Signup/Login)
- 📸 Create, upload, and delete posts  
- ❤️ Like and interact with posts  
- 💬 Comment system  
- 🧑 User profile page  
- 📰 Feed with real-time updates  
- 🌐 Responsive design for all devices  

---

## 🛠️ Tech Stack

- **Frontend:** React.js  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Authentication:** JWT  
- **Deployment:** Render  

---


## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/vivekchanne06-web/Insta-Clone.git

# Navigate to project folder
cd Insta-Clone

# Install backend dependencies
npm install

# Navigate to frontend (if separate folder)
cd client
npm install

# Run backend
npm run server

# Run frontend
npm start
```
